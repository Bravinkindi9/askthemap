\# AskTheMap



Ask any question about any point on Earth and get an answer grounded in satellite imagery.



\## Problem



Understanding places on Earth usually requires GIS skills, expensive tools, or technical knowledge.



AskTheMap makes geospatial intelligence accessible through natural language.



\## User



Students, researchers, planners, journalists, developers, and curious people who want to understand locations using satellite imagery.



\## V1 Goal



A web application where a user can:



\- select a location

\- ask a question

\- receive an AI-generated answer based on satellite imagery
